## 명령어 실행 과정

### 1. 요청

```

```

Claude가 확인을 요청합니다:

```

> npm install express

[Y/n]
```


## 권한 관리

### 명령어별 자동 승인

자주 쓰는 명령어는 자동 승인할 수 있습니다:

```
/allow Bash(npm:*)
```

이제 npm 명령어는 확인 없이 실행됩니다.

### 설정 파일

`~/.claude/settings.json`:

```json
{
  "permissions": {
    "allow": [
      "Bash(npm:*)",
      "Bash(git:*)",
      "Bash(ls:*)"
    ]
  }
}
```
