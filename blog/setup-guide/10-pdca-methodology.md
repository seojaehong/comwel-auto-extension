## PDCA란?

**PDCA**는 지속적 개선을 위한 사이클입니다:

```
Plan (계획) → Do (실행) → Check (검증) → Act (조치)
     ↑                                      ↓
     ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
```

원래 품질 관리에서 시작된 방법론인데, 소프트웨어 개발에도 적용됩니다.

---

## bkit의 PDCA

bkit은 PDCA를 개발에 맞게 확장했습니다:

### Plan (계획)
- 요구사항 분석
- 스키마/용어 정의
- 기술 스택 결정

### Design (설계)
- API 설계
- UI/UX 설계
- 데이터 모델 설계

### Do (구현)
- 코드 작성
- 테스트 작성
- 통합

### Check (검증)
- 갭 분석 (설계 vs 구현)
- 코드 품질 분석
- 테스트 실행

### Act (조치)
- 문제 수정
- 개선 반복
- 문서화

## PDCA 스킬 사용법

### /pdca plan

```
/pdca plan 로그인 기능
```

계획 문서를 생성합니다:
- `docs/01-plan/login-plan.md`

### /pdca design

```
/pdca design 로그인 기능
```

설계 문서를 생성합니다:
- `docs/02-design/login-design.md`

### /pdca do

```
/pdca do 로그인 기능
```

구현을 시작합니다.

### /pdca analyze

```
/pdca analyze 로그인 기능
```

갭 분석을 수행합니다:
- 설계 문서와 구현 코드 비교
- 매칭률 계산
- 누락/불일치 항목 식별

### /pdca report

```
/pdca report 로그인 기능
```

완료 리포트를 생성합니다:
- `docs/04-report/login-report.md`

## 갭 분석 (Check)

### 갭 분석이란?

**설계 문서**와 **실제 구현**을 비교해서 차이를 찾는 것

### 예시

**설계서:**
```markdown
## 로그인 API
- POST /api/login
- 요청: { email, password }
- 응답: { token, user }
- 에러 처리: 401, 400
```

**실제 구현:**
```javascript
// POST /api/login 구현됨 ✓
// 요청 형식 일치 ✓
// 응답에 user 없음 ✗
// 에러 처리 미구현 ✗
```

**갭 분석 결과:**
```
매칭률: 60%
누락 항목:
- 응답에 user 객체
- 에러 처리 (401, 400)
```

### 자동 갭 분석

```
/pdca analyze 로그인 기능
```

또는 자연어로:

```
로그인 기능 설계서랑 구현 비교해줘
```

## PDCA 반복

### 90% 규칙

bkit은 **매칭률 90%** 이상을 목표로 합니다.

- 90% 미만: 자동 개선 제안
- 90% 이상: 완료 처리

### 자동 반복 개선

```
/pdca iterate 로그인 기능
```

매칭률이 90%가 될 때까지 자동으로:
1. 갭 분석
2. 코드 수정
3. 재분석
4. 반복...

## 실제 예시

### 1. 계획

```
/pdca plan 사용자 프로필 페이지
```

생성된 문서:
```markdown
# 사용자 프로필 페이지 계획

## 목표
사용자가 자신의 프로필을 조회/수정할 수 있는 페이지

## 기능
- 프로필 조회
- 프로필 수정
- 프로필 사진 업로드

## 기술 스택
- React
- REST API
```

### 2. 설계

```
/pdca design 사용자 프로필 페이지
```

### 3. 구현

```
/pdca do 사용자 프로필 페이지
```

### 4. 검증

```
/pdca analyze 사용자 프로필 페이지
```

결과:
```
매칭률: 75%
누락: 프로필 사진 업로드
```

### 5. 조치

```
프로필 사진 업로드 기능 추가해줘
```

### 6. 재검증

```
/pdca analyze 사용자 프로필 페이지
```

결과:
```
매칭률: 95% ✓
```

### 7. 리포트

```
/pdca report 사용자 프로필 페이지
```
