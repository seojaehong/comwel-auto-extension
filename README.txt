======================================
근로복지공단 자동화 Chrome 확장프로그램
======================================
버전: 1.0.0
대상: total.comwel.or.kr

[설치 방법]
-----------

1. 아이콘 생성 (최초 1회)
   - icons/generate-icons.html 파일을 브라우저에서 열기
   - 각 사이즈별 다운로드 링크 클릭
   - 다운로드된 파일을 icons 폴더에 저장

2. Chrome 확장프로그램 로드
   - Chrome 브라우저에서 chrome://extensions 접속
   - 우측 상단 "개발자 모드" 활성화
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - comwel-auto-extension 폴더 선택

3. 설치 확인
   - 브라우저 우측 상단에 확장프로그램 아이콘 표시
   - 아이콘 클릭하여 팝업 확인


[기능 사용법]
-------------

1. 로그인 자동화
   - 팝업에서 아이디/비밀번호 입력
   - "로그인 정보 저장" 체크하면 암호화 저장
   - "로그인 실행" 클릭
   - 근로복지공단 로그인 페이지에서 작동

2. 데이터 추출
   - 근로복지공단 사이트 접속
   - 팝업의 "데이터" 탭 선택
   - 원하는 추출 버튼 클릭 (테이블/폼/전체)
   - JSON 또는 CSV로 다운로드

3. 양식 자동 입력
   - "양식입력" 탭 선택
   - JSON 형식으로 필드:값 입력
   - 예: {"userId": "hong", "name": "홍길동"}
   - "양식 자동 입력" 클릭

4. 일괄 작업
   - "일괄작업" 탭 선택
   - 작업 배열을 JSON으로 입력
   - 또는 JSON/CSV 파일 불러오기
   - "일괄 작업 실행" 클릭

   작업 유형:
   - fill: 폼 입력 {"type": "fill", "data": {...}}
   - click: 버튼 클릭 {"type": "click", "selector": "#btnId"}
   - wait: 대기 {"type": "wait", "duration": 1000}
   - extract: 데이터 추출 {"type": "extract", "dataType": "table"}


[설정]
------
- 자동 로그인: 사이트 접속 시 자동 로그인
- 알림 표시: 작업 완료 알림
- 작업 간 대기 시간: 일괄 작업 시 대기 시간(ms)


[보안 안내]
-----------
- 비밀번호는 암호화되어 로컬에만 저장됩니다
- 외부 서버로 전송되지 않습니다
- 민감한 정보는 신중하게 관리하세요


[문제 해결]
-----------

Q: 로그인이 안 됩니다
A: 근로복지공단 로그인 페이지에 접속 후 시도하세요.
   사이트 구조가 변경된 경우 셀렉터 수정이 필요할 수 있습니다.

Q: 데이터 추출이 안 됩니다
A: 추출하려는 페이지가 완전히 로드될 때까지 기다린 후 시도하세요.

Q: 확장프로그램이 작동하지 않습니다
A: chrome://extensions에서 확장프로그램을 다시 로드하세요.

[WebSquare 호환성]
------------------
이 확장프로그램은 WebSquare 프레임워크 기반 사이트에서
작동하도록 설계되었습니다. WebSquare 컴포넌트의 값 설정과
이벤트 트리거를 지원합니다.
